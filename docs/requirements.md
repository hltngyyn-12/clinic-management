# Requirements

## 1. Project Overview
Clinic Management System is a web-based system that helps patients book appointments online, doctors manage examination schedules, and administrators manage clinic resources.

## 2. Main Roles
### Patient
- Register and login
- View doctors and specialties
- Book appointments
- View appointment history
- View prescriptions
- View test results
- Review doctor

### Doctor
- View daily appointments
- View patient information
- Create medical records
- Prescribe medicines
- Request tests
- View patient medical history

### Admin
- Manage doctors
- Manage specialties
- Manage medicines
- Configure appointment slots
- View reports

## 3. Functional Requirements
- Authentication and authorization
- Appointment booking management
- Medical record management
- Prescription management
- Test result management
- Basic report management

## 4. Non-functional Requirements
- Easy to use
- Secure login and role-based access
- Responsive web interface
- Maintainable source code structure

## 5. Scope of Version 1
- 3 roles
- 15-20 features
- 6-10 database tables
- 20-30 API endpoints
- Full CRUD for at least 3 entities
